StreamlineCloud is a free and opensource cloud system for paper and velocity based minecraft networks. This documentation will help you get started and make the most of StreamlineCloud's features.

## Quick start
If you are using a Linux based system you can use our quick start script
```bash
curl https://streamlinecloud.net/quick-start.sh
```
